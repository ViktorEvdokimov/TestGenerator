TestGenerator - программа для генерации JSON файла тестовых наборов групп случайных людей. Каждая группа 
имеет уникальное название и случайное количество человек. Каждый человек имеет ФИО, адрес, номер телефона и GUID.
Названию группы, фамилии, имени, отчеству и улице присваивается случайная строка из файлов с адресом 
"src\main\resources\Data".
Для запуска программы в командной строке перейдите по адресу корневой папки проекта (к примеру командами "D:" и 
"cd D:\Projects\TestGenerator") далее введите команду:
java -jar TestGenerator-1-jar-with-dependencies.jar 
при этом в корневой папке будет создан файл или обновлен (если уже существует) "groups.json" который будет содержать 
25 групп и 500 человек (случайным образом распределенных по этим группам). Для изменения количества групп и количества 
человек используйте аргументы:
-gc [положительное целое число, меньше или равное количеству строк в файле src\main\resources\Data\groups.txt] - количество 
групп.
-pc [положительное целое число] - количество человек.
Пример команды с аргументами:
java -jar TestGenerator-1-jar-with-dependencies.jar -gc 50 -pc 1000
